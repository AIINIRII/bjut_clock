# bjut_clock

北工大疫情自动打卡脚本（北京工业大学  疫情）

## 脚本三部分

### 自动登录

只需要将其中username和password修改成自己的就可以了

### 获取昨天数据

其中获得的数据是个json串的
    {
        e:0
        m:"操作成功"
        d:{
        ...
        }
    }
其中要提交的数据都在 d 中。将其解析返回。

### 对数据进行包装提交

剩下的就是重新进行封装提交了

## 注：若是北工大的，只需要改下登录那块账号密码就可以使用了

其中可能还有输入的，放到服务器改成日志就可以了~
